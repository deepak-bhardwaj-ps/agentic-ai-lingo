Bolt is a browser-based tool that helps you build websites and web apps by typing what you want in plain English.

In practice, you describe the app, and Bolt can draft code, show a live preview, and help you keep editing it in the browser. It is meant for making software faster, especially for people who want to go from an idea to something they can test quickly.

The term matters because it names a specific product, not a general idea. People use Bolt when they want a prompt-to-app tool that runs in the browser and helps build real web software without starting from a blank local setup.

It is not a general AI model, and it is not the same thing as the open-source codebase behind it. It is also not a magic button that makes finished software with no checking; you still need to review what it builds.
