Quantitative finance agent is an agent that helps with data-heavy investing work, especially trading or portfolio research based on numbers, prices, and market patterns.

In practice, it may read market data, news, charts, and financial reports, then suggest trades, rankings, or portfolio changes. Some versions use several smaller agents, each with a different job such as spotting trends, judging risk, or checking signals.

The term matters because quantitative finance uses lots of data and repeated rules, so an agent can help people sort information faster and test ideas more quickly.

It is not a general finance chatbot, and it is not the same as a normal trading algorithm. The term is also not fully standard, so different teams may use it in slightly different ways.
