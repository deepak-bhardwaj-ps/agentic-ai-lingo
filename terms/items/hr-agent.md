HR agent is not a single fixed standard term. In practice, it usually means an AI agent used for human-resources work.

In practice, an HR agent helps people with tasks like answering policy questions, guiding onboarding, handling routine requests, or moving HR work through a system. It may talk in plain language, but it is meant to do more than just chat.

This term matters because HR teams often spend a lot of time on repeated questions and admin work. An HR agent can reduce that load and help employees get faster answers.

It is not a human HR employee, and it is not the same as a general chatbot. Some HR agents only answer questions, while others can also start workflows or complete tasks. The term is still being used in different ways, so the exact meaning depends on the product or company using it.
