## Term status

Emerging practitioner shorthand.

## Meaning

Context Operating System concerns the selection, quality, lineage or delivery of information presented to an agent at runtime.

## Boundary

It is not a defined product category. Make the retrieval, freshness, ownership, permissions and token-budget rules explicit.

## How it is used

Context Operating System is used when context handling has become a first-class runtime concern with rules, policies, and shared services. It is the strongest label in this family, but it only earns the name if the system actually orchestrates context as a managed layer.

## Evidence

[Lewis et al., Retrieval-Augmented Generation](https://arxiv.org/abs/2005.11401) provides the relevant primary source or established reference. For coined labels, it is background for the underlying concept—not evidence that the label itself is standard.
