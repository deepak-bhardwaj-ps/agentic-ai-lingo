Backlinks are links from other notes that point back to the note you are looking at.

In practice, if note A links to note B, then note B has a backlink from note A. Many wiki and note-taking tools show these in a side panel so you can see where a note is mentioned or referenced.

Backlinks matter because they help you find related notes, follow ideas in both directions, and understand where a fact or claim came from.

Backlinks are not the same as tags, and they are not the same as unlinked mentions. A tag groups notes by a label. A backlink is a real link created by one note pointing to another.
