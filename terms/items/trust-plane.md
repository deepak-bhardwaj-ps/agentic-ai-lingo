## Term status

Governance/security concept.

## Meaning

Trust Plane concerns how an agent's authority, obligations, actions or oversight are defined and evidenced.

## Boundary

It is not a control merely because it is named. Define the accountable principal, permitted scope, enforcement point, audit record, revocation and human escalation.

## How it is used

Trust Plane is used when trust assessment is handled in a distinct operational layer. It only matters if the layer changes what the system may do.
