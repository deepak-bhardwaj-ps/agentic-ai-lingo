Anthropic Claude Code is Anthropic's name for Claude Code, an agentic coding tool.

In practice, it is a tool that can read your codebase, change files, run commands, and help with coding tasks. It is used from places like the terminal, an IDE, a desktop app, or a browser, depending on the setup.

The term matters because it tells you this is a product, not just a model name. Claude Code is designed to work across many files and tools, so it can do more than chat about code.

It is not the same thing as the Claude model itself. It is also not a general term for every AI coding assistant.
