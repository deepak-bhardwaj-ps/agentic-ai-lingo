Bio agent is an emerging term for an AI agent used in biology or biomedical work.

In practice, it means a system that can help with tasks like searching papers, analysing biological data, planning research steps, or running tools for a scientist.

The term matters because biology work can involve large amounts of data and many small steps. A bio agent can help organise that work and save time, but it still needs human checking.

It is not a biological agent, such as a virus or bacterium. It is also not just a chatbot that talks about biology without using tools or working through a task.
