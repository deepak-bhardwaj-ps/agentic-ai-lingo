Warp is a developer terminal app.

In practice, that means you use Warp to type commands, run tools, and work with code from the command line, but with extra features that make the terminal easier to use. Those features include things like better editing, saved command workflows, and built-in AI help.

Warp matters because it tries to make terminal work less clumsy for developers. It keeps the command line, but adds tools that can help people move faster and make fewer mistakes.

Warp is not a general word for any AI system. It is also not the same as the normal word “warp”. In this glossary, Warp means the specific developer product made by Warp.
