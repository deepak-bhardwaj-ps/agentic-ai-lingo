Working memory is the small, temporary set of information an agent is using right now.

In practice, it is the active context: the recent messages, notes, tool results, and plan that the model can still pay attention to while it works. If the task is long, older details may be trimmed, summarised, or moved elsewhere so the agent can stay focused.

This matters because an agent cannot keep everything in its head at once. Working memory helps it follow the current task without mixing in old or irrelevant details.

It is not long-term memory. It is not the model’s permanent memory, and it is not the same as saving facts in a database or profile for later use.
