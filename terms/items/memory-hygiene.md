Memory hygiene is the routine care of an agent's stored memories so they stay accurate, current, and safe to use.

In practice, it means checking what the agent remembers, removing stale or wrong facts, merging duplicates, and keeping track of where each memory came from. It also means deciding what should be kept for a long time and what should be thrown away.

This matters because an agent can only behave well if its memory is clean. Old or duplicated memories can confuse it, waste space, and make it give the wrong answer.

It is not the same as training the model itself. It is also not just “having memory”. Memory hygiene is the work of maintaining that memory after it has been stored.
