Comet is Perplexity's web browser with built-in AI help.

In practice, it lets you browse the web, ask questions about what is on a page, and get help with tasks like summarising, researching, shopping, and managing email.

It matters because it changes the browser from a place where you only look at web pages into a place where software can help you do work inside those pages.

It is not a general word for any browser with AI features, and it is not a human helper. It is Perplexity's specific product, and some actions still need you to check and approve them.
