Dia is The Browser Company’s AI browser.

In practice, it is a normal web browser with an assistant built in. You can ask it about the page you are on, other open tabs, your browser history, or selected text. It also keeps some memory so it can help you continue work later.

It matters because it tries to turn the browser from a place where you only read web pages into a place where you can also ask for help, summarise information, and complete simple work faster.

It is not a general name for any browser with AI features. It is a specific product. It is also not a human helper, and it should not be treated as if it can safely do every action without checking.
