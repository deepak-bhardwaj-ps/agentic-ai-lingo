Codex is OpenAI's coding agent for software development.

In practice, that means Codex helps with coding tasks such as writing code, reading a codebase, reviewing changes, finding bugs, and running repetitive development work. It can be used in a terminal, inside an editor, or in the cloud.

The term matters because it is not just a chatbot with a coding prompt. It is a product name for a set of developer tools that work with real code and developer workflows.

It is not a general term for every AI coding helper, and it is not the old OpenAI text model name from earlier years. In this glossary, Codex means the current OpenAI product family for coding work.
