OpenCode is an open-source AI coding agent.

It helps people work on software by using tools in a codebase, such as reading files and making changes, from places like a terminal, desktop app, or IDE extension.

In practice, it is used to help with coding tasks such as exploring a project, planning changes, and doing development work inside the tools a programmer already uses.

This term matters because it refers to a specific product, not a general idea. OpenCode is a named tool with its own features and interfaces.

It is not the same thing as the phrase “open code”, and it is not every AI coding tool. It is specifically the OpenCode product.
