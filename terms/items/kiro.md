Kiro is an AI coding tool from AWS.

It helps you turn a prompt or a rough idea into a more detailed plan, then into code, tests, and documentation. In practice, it is more than a chat box: it is an IDE, so you can work inside a code editor while the tool helps organise the job, track changes, and support agent-style coding.

Kiro matters because it tries to make AI coding more structured. Instead of only asking for code line by line, you can describe what you want, shape it into a spec, and use that spec to guide the work.

It is not a general word for all AI coding assistants. It is also not the same thing as Amazon Q Developer, even though both are AWS developer tools.
