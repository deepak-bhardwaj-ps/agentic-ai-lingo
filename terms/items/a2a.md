## Term status

Established protocol.

## Meaning

An open protocol for agent-to-agent communication. It defines discovery, capability advertisement, task lifecycle, messages, artefacts and streaming between independently built agents.

## Boundary

It is not a multi-agent framework and does not specify an agent's reasoning loop, tools or safety policy.

## How it is used

A2A is used when the question is how one agent discovers, addresses, and delegates work to another. If the term is being used seriously, it should come with concrete message shapes, authentication, compatibility, and failure behaviour rather than just “agent-to-agent” branding.
