An agent coalition is a group of separate agents that choose to work together on one goal.

In practice, each agent still has its own job, limits, and rules. They do not become one single agent. They may pass work to each other, split up tasks, or combine their results.

The term matters because real systems are often built from parts made by different teams or companies. A coalition is a useful way to describe agents that cooperate without being merged into one system.

It is not a formal standard that everyone uses in the same way. It is also not the same as one app controlling a set of sub-agents. When people use the phrase, they usually mean coordinated agents rather than a strict technical rule.
