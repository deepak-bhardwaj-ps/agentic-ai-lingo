CodeGPT is a coding assistant product.

It is used inside code editors such as VS Code and JetBrains to help with tasks like writing code, changing files, explaining code, and fixing bugs.

In practice, that means you can ask it to help with a coding job and it can work across one or more files instead of only answering questions.

The term matters because CodeGPT is not just a chat box. It is a tool for doing coding work inside the place where developers already work.

It is not a general name for every coding assistant. It is also not the same as the older “Code GPT” extension, which was a narrower VS Code tool.
