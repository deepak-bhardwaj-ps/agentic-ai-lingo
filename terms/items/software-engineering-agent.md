Software engineering agent is an AI agent built to help with software work.

In practice, it can read a codebase, make edits, run commands or tests, and sometimes open a pull request for a person to review.

The term matters because these tools do more than suggest text. They can carry out parts of the software development process, so they need clear tasks, checks, and human review.

It is not just code autocomplete. It is also not the same as a general chatbot, and it is not a full replacement for a human software engineer.
