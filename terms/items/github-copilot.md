GitHub Copilot is GitHub’s coding assistant. It suggests code, answers coding questions, and helps with development tasks inside supported editors and on GitHub.

In practice, it can show code suggestions while you type, explain code, help write tests, and answer questions about code. It is meant to speed up software work, not to replace the programmer.

The term matters because GitHub Copilot is one of the best-known tools in coding assistance. It changed what many people expect an editor helper to do: not just finish a line, but also help with debugging, explanations, and other tasks.

It is not a human programmer, and it is not a general-purpose chatbot. It is also not the same thing as other products that use the word "copilot". In this glossary, the term refers to GitHub’s product specifically.
