Junie is JetBrains' coding agent.

In practice, Junie helps with programming work inside JetBrains tools. It can plan steps, make code changes, run checks, and help review work. JetBrains also describes it as available in the IDE, the terminal, and CI/CD flows.

It matters because it turns some programming tasks into instructions you can delegate instead of doing every step by hand.

It is not a general chat bot, and it is not the same thing as JetBrains AI Assistant, which is a related but separate tool.
