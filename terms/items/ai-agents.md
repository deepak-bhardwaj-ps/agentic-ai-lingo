## Term status

Umbrella term.

## Meaning

AI agent is an overloaded label for a model-based system that selects actions towards a goal, commonly by invoking tools and using observations. The autonomy and planning depth vary widely.

## Boundary

A chatbot, workflow and autonomous agent are not interchangeable. State the action surface, stopping condition and human approval points.

## How it is used

AI Agents is used when the discussion is about a system that can plan, call tools, track state, and recover from failure rather than only answer a prompt. In practice it is the broadest umbrella in the glossary, so it should be narrowed quickly to the real runtime, governance, or workflow question.
